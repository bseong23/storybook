# Markup UI Component Project

통일된 UI 컴포넌트(디자인 시스템)를 제작하고, 해당 UI 컴포넌트를 사용하여 직접 오픈소스 웹사이트 HTML 템플릿을 구현합니다.

- 엘리스 SW 트랙 9기 1차 스터디 프로젝트.
  - **팀장** [김민지](https://github.com/mjae404)
  - **팀원** 정경준, 최하은, 박보성
